## Data Details

*   **RGB Panorama (`panoImage_1600.jpg`)**:
    *   **Real-world data**: This is a high-quality color panoramic image synthesized from multiple high-definition fisheye lens images. Due to the vertical FOV limitations of the acquisition equipment, there are blind spots in the north and south polar regions of the panorama. `pano_mask.png` provides detailed information about these blind areas.
        *   Example of a real-world RGB panorama: ![Real-world RGB panorama](assets/panoImage_real.jpg)
        *   Example of `pano_mask` visualization (red indicates valid regions): ![Pano Mask Visualization](assets/panoImage_mask_vis.jpg)
    *   **Virtual data**: The color panoramic image is generated by a rendering engine and has no blind spots. Therefore, `pano_mask.png` files are not provided for procedurally generated scenes.
        *   Example of a virtual RGB panorama: ![Virtual RGB panorama](assets/panoImage_synthesis.jpg)
*   **Depth Map (`depth_image.png`)**: 16-bit depth map aligned with the RGB panorama.
*   **Extrinsics (`extrinsics.txt`)**: The pose of the camera for the viewpoint.
*   **Depth Scale (`depth_scale.txt`)**: The divisor used to obtain the metric depth (e.g., `depth_in_meters = pixel_value / scale`).
*   **Floor Info (`floor.txt`)**: Parameters defining the floor plane.
